# Sistema de Gestão de Sala de Aula Online

Projeto desenvolvido como requisito para a Avaliação Integradora da disciplina de Modelagem e Projetos em Engenharia de Software.

## Visão Geral
Este repositório contém toda a documentação, modelagem e arquitetura para a concepção de uma plataforma digital voltada para a gestão de turmas, atividades, avaliações e comunicação entre professores e estudantes, inspirada no Google Classroom.

## Estrutura do Repositório

- `/docs`: Documentação textual do projeto (Visão do Produto, Regras de Negócio, User Stories, MVP, Arquitetura e Diário de Sprint).
- `/bpmn`: Modelagem do processo de negócio (Diagrama de Entrega de Atividade).
- `/uml`: Diagrama de Casos de Uso.
- `/c4`: Modelagem arquitetural (Diagramas de Contexto e Containers).
- `/apresentacao`: Arquivos da apresentação final para a banca.

## Funcionalidade Inovadora
**Sistema de Gamificação (XP e Níveis):** Os estudantes recebem pontos de experiência automaticamente ao entregarem atividades no prazo. Esta funcionalidade visa aumentar o engajamento e combater a evasão e o atraso nas entregas, utilizando mecânicas de jogos aplicadas à educação. A solução foi projetada de forma desacoplada (via mensageria) para não impactar o desempenho do sistema principal.
